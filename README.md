# 冷库 3D 温湿度监控看板

一个纯前端的冷库 3D 可视化监控看板，用于直观展示冷库内各货架货位的温湿度传感器状态。

## 功能特性

- 🏭 3D 冷库场景可视化（4 排货架 × 5 个货位 × 3 层高度）
- 🎯 传感器点位按状态着色（正常/偏冷/偏热/湿度偏高/离线）
- 🔄 三种着色模式切换（按温度 / 按湿度 / 按电量）
- 🔍 左侧过滤面板（按状态、货品类型、货架排号）
- 📊 右侧详情面板（点位信息 + 最近 6 次读数折线图）
- 🔥 俯视热区小图（异常密度热力图）
- 📈 实时统计（总点位、异常、离线、平均温度、最高湿度）
- 💾 导出异常清单（JSON 格式）
- 🎮 3D 交互（旋转、缩放、平移）

## 技术栈

- **Three.js** - 3D 渲染引擎
- **Chart.js** - 图表库
- **Vite** - 构建工具

## 样例数据规模

- **总点位数量**：60 个
  - 4 排货架 × 5 个货位 × 3 层高度 = 60 个传感器点位

## 状态分布说明

每次刷新页面数据会随机生成，大致分布如下：

| 状态     | 大约数量 | 说明                                   |
| -------- | -------- | -------------------------------------- |
| 正常     | ~40 个   | 温湿度均在正常范围内                   |
| 偏热     | ~5 个    | 温度高于货品设定温区上限               |
| 偏冷     | ~5 个    | 温度低于货品设定温区下限               |
| 湿度偏高 | ~5 个    | 湿度高于货品设定湿度上限               |
| 离线     | ~5 个    | 最后上报时间超过 30 分钟               |
| 低电量   | ~5-8 个  | 电量低于 20%（在按电量着色模式下可见） |

> **注意**：左侧面板分为两个列表：
>
> - **异常点位列表**：显示偏热、偏冷、湿度偏高的点位（与顶部"异常"统计一致）
> - **离线点位列表**：显示离线的点位（与顶部"离线"统计一致）

## 货品类型

共 6 种货品类型，每种有不同的温湿度要求：

| 货品类型 | 温度范围      | 湿度范围  |
| -------- | ------------- | --------- |
| 冷冻肉类 | -25°C ~ -18°C | 60% ~ 85% |
| 海鲜水产 | -22°C ~ -15°C | 65% ~ 85% |
| 果蔬保鲜 | 2°C ~ 8°C     | 75% ~ 90% |
| 乳制品   | 2°C ~ 6°C     | 60% ~ 80% |
| 医药疫苗 | 2°C ~ 8°C     | 40% ~ 60% |
| 冰淇淋   | -28°C ~ -22°C | 50% ~ 70% |

由于数据随机生成，以下点位大概率存在异常（按位置查找）：

### 第 3 排

- **4 货位 2 层** - 偏热

## 配置说明

异常判断规则配置在 `src/config/index.js` 文件中：

- **离线阈值**：`offlineThresholdMinutes` - 默认 30 分钟
- **低电量阈值**：`lowBatteryThreshold` - 默认 20%
- **高湿度阈值**：`highHumidityThreshold` - 默认 85%
- **各货品温区**：`productTypes` - 可自行调整每种货品的温湿度范围

## 操作说明

### 3D 场景操作

- **左键拖动**：旋转视角
- **右键拖动**：平移视角
- **滚轮**：缩放

### 界面操作

- 点击顶部「按温度/按湿度/按电量」切换着色模式
- 左侧面板勾选状态、货品类型、货架排号进行筛选
- 点击左侧异常列表中的条目，镜头会自动聚焦到对应传感器
- 点击 3D 场景中的传感器点位，右侧显示详细信息
- 点击左下角俯视热区图的格子，筛选该排该层的点位
- 点击右上角「导出异常清单」下载 JSON 文件
